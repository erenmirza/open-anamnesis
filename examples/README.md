# Example Projects

This directory contains example Anamnesis projects that demonstrate the platform's features.

## Running Examples

### 1. Basic Project

The simplest example with one deck:

```bash
cd basic_project
anamnesis-compile    # Validate
anamnesis-build      # Build and run
```

### 2. Multi-Deck Project

Example with multiple decks and dependencies:

```bash
cd multi_deck_project
anamnesis-compile
anamnesis-build
```

## Creating Your Own

Use `anamnesis init` to create a new project:

```bash
anamnesis init my_project
cd my_project
```

Then follow the structure shown in these examples.
